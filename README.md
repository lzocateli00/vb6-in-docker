## Running

This container is to be run on Windows

To start a container in detached mode:

````pwsh
docker run --name vb6-in-docker `
    -d `
    lzocateli/vb6-in-docker:1.0.0 
````

To start a container in foreground mode:

````pwsh
docker run --name vb6-in-docker `
    -ti `
    --rm `
    lzocateli/vb6-in-docker:1.0.0 
````
